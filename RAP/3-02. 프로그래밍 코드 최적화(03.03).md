## <  핵심 개념 >
- **1) RAP 기반 CDS → OData → Fiori Elements 전체 구현 전체 개발 절차**
- **2) UI 어노테이션 그룹핑(Grouping)**
- **3) Metadata Extension - List Report 컬럼과 필터 설정 어노테이션**
- **4) Metadata Extension - `@UI.hidden` 어노테이션**
- **5) Metadata Extension - `@UI.headerInfo` 화면 상단 정보 설정**
- **6) BAS - Fiori List Report – View 저장 및 관리 기능**
- **7) Projection View - 검색 기능 설정 어노테이션 (`@Search.*`)**
- **8) Metadata Extension - `@Consumption.filter` 필터 동작 설정**
- **9) BAS - `Manifest.json` 오브젝트 페이지 설정 제거**
- **10) BAS - 메인 앱 타이틀 변경 방법**
- **11) Projection View - Value Help(F4) 연결 설정**
- **12) Projection View - 코드-텍스트 연결 및 표시 순서 설정**
- **13) 메타데이터에 정의하지 않은 필드의 Fiori 표시 동작**
