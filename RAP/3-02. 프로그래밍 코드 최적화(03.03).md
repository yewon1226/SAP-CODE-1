### 3) Search 기능을 포함한 Value Help CDS 코드 구현
```abap
Data Definitions - ZVH_CARRIER_B03

@AbapCatalog.viewEnhancementCategory: [#NONE]
@AccessControl.authorizationCheck: #NOT_REQUIRED
@EndUserText.label: 'Value Help Carrid'
@Metadata.ignorePropagatedAnnotations: true
@Search.searchable: true
@ObjectModel.resultSet.sizeCategory: #XS
define view entity ZVH_CARRIER_B03 as select from /DMO/I_Carrier
{
    @Search.defaultSearchElement: true -- SearchBar 검색 대상
    @Search.fuzzinessThreshold: 0.8
    @ObjectModel.text.element: [ 'Name' ]
    @UI.textArrangement: #TEXT_FIRST
    key AirlineID,
    
    @Search.defaultSearchElement: true -- SearchBar 검색 대상
    @Search.fuzzinessThreshold: 0.8
    Name,
    
    @Search.defaultSearchElement: true -- SearchBar 검색 대상
    @Search.fuzzinessThreshold: 0.8
    CurrencyCode
}
```

### 5) 인터페이스 루트 뷰에서 아이템 Composition 정의

```abap
define root view entity ZDSFLIGHT_IR_B03 as select from sflight
```

## < CDS 주요 어노테이션 기능 정리 >
</br>

### 1) CDS Value Help usage 옵션 기능
- `usage` 는 Value Help에서 필터로 쓸지, 선택값을 되돌려줄지 동작 방식을 정하는 옵션
- 미지정 (= `#FILTER`) → WHERE <VH필드> = <현재값> 로 목록만 필터됨
-  `#RESULT` → <현재필드> = <선택값> 으로 선택 후 자동세팅됨
-  `#FILTER_AND_RESULT` → WHERE … + <현재필드> = <선택값> 둘 다 수행됨

```abap
@AccessControl.authorizationCheck: #NOT_REQUIRED

@Metadata.ignorePropagatedAnnotations: false

    key Carrid,
```
