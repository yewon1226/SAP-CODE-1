## < ABAP Editor Code Completion >
- `CTRL + Space` : 코드 자동완성 호출
- `Shift + Enter` Or `CTRL + Space + Enter` : 선택한 메서드 템플릿 자동 삽입
- 'CTRL + `' : 터미널 창 열기
- `npm run start` : 실행
