# ERD

### < ERD (Entity Relationship Diagram) >
- 테이블 구조를 시각적으로 파악하기 위함
- 테이블 간의 관계를 나타내기 위함

<img width="435" height="295" alt="image" src="https://github.com/user-attachments/assets/e81e8053-b671-49ae-b1d9-6d138cd4655a" />

- ONE : 하나의 A는 하나의 B로 연관
- MANY : 하나의 A는 여러 개의 B로 연관
- One or Many : 하나의 A는 1개 또는 여러 개의 B로 연관
