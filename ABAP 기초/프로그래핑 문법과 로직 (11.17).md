### ABAP 타입 3종
- `DATA`는 메모리와 값을 갖는 변수 생성, `TYPE`은 값 없이 타입만 정의
1) Data Element Type
   - 하나의 값(필드 하나)을 표현하는 타입
   - 길이/형식 같은 기본 속성을 정의함

2) Structure Type
   - 여러 필드가 모여 하나의 덩어리를 이루는 타입
   - 하나의 레코드(행) 같은 구조
  
3) Table Type
   - 동일한 구조를 여러 개 담는 “내부 테이블”을 위한 타입
   - 여러 행을 저장하는 컬렉션 형태

### 데이터 요소(Data Element) 타입으로 올 수 있는 3가지 종류
#### 1) Standard Data Type
   - SAP가 제공하는 기본 데이터 타입
      #### 1-1. Complete Type
      | Type       | 설명                                                                 |
      |------------|----------------------------------------------------------------------|
      | D          | 날짜(Date)                                                             |
      | T          | 시간(Time)                                                             |
      | I          | 4바이트 정수(Integer 4)                                               |
      | INT8       | 8바이트 정수(Integer 8)                                               |
      | F          | 부동 소수점(Float)                                                     |
      | STRING     | 가변 길이 문자열(Variable-length string), 사용자가 입력하는 만큼 동적 할당됨 |
      | XSTRING    | 가변 길이 이진(Binary) 문자열, 사용자가 입력하는 만큼 동적 할당됨      |
      | DECFLOAT16 | 16자리 부동 소수점(decimal floating)                                  |
      | DECFLOAT34 | 34자리 부동 소수점(decimal floating)                                  |
      
      #### 1-2. Incomplete Type
      | Type | 설명                                                      |
      |------|-----------------------------------------------------------|
      | C    | 문자(Character), 길이 지정 필요                            |
      | N    | 숫자 문자(Numeric character), 길이 지정 필요              |
      | X    | 16진수(Hexadecimal), 길이 지정 필요                       |
      | P    | Packed Decimal(BCD), 소수점 위치 지정 필요                 |
      
      
#### 2) Local Type
   - 특정 프로그램 내에서 정의하여 사용하는 타입
   - 다른 프로그램에서는 사용할 수 없음
  
#### 3) Global Data Type
   - 동일한 구조를 여러 개 담는 “내부 테이블”을 위한 타입
   - 여러 행을 저장하는 컬렉션 형태
     
data element 타입으로 올 수 있는 3가지 종류
1. standard data type   : 기본 데이터타입
   1-1. complete type   : 자릿수가 이미 지정됨
   1-2. incomplete type   : 자릿수 지정 필요
2. local type : 프로그램 내에서 정의하여 사용
3. global data type   : 모든 프로그램에서 사용 (SE11)
