- 실습 1
```abap
PARAMETERS pa_num TYPE I.
DATA gv_result TYPE I.

IF ( pa_num <= 1 ).
  MESSAGE '2 이상을 입력하세요' TYPE 'E'.
ELSEIF ( pa_num >= 10 ).
  MESSAGE '9 이하를 입력하세요' TYPE 'E'.
ELSE.
  DO 9 TIMES.
    gv_result = pa_num * sy-index.
    WRITE:/ pa_num, ' * ', sy-index, ' = ', gv_result.
  ENDDO.
ENDIF.
```
