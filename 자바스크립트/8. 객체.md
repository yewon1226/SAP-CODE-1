# 객체
### 객체 (1/3) : 객체의 문법
</br>

### < 객체의 생성 >
```java
var grades = {'egoing': 10, 'k8805': 6, 'sorialgi': 80};
```
- egoing은 key(배열에서는 인덱스), 10은 value(배열에서는 값)
```java
var grades = {};
grades['egoing'] = 10;
grades['k8805'] = 6;
grades['sorialgi'] = 80;
```
```java
var grades = new Object();
grades['egoing'] = 10;
grades['k8805'] = 6;
grades['sorialgi'] = 80;
```
- 위와 같은 객체를 만드는 다른 방법
</br>
</br>

```java
var grades = {'egoing': 10, 'k8805': 6, 'sorialgi': 80};
alert(grades['sorialgi']);
```
- 80 출력
```java
alert(grades.sorialgi);
alert(grades['sorial'+'gi']);
```
- 위와 같은 객체의 속성에 접근하는 다른 방법
