# 객체
### 객체 (1/3) : 객체의 문법
</br>

### < 객체의 생성 >
```java
var grades = {'egoing': 10, 'k8805': 6, 'sorialgi': 80};
```
- egoing은 key(배열에서는 인덱스), 10은 value(배열에서는 값)
```java
var grades = {};
grades['egoing'] = 10;
grades['k8805'] = 6;
grades['sorialgi'] = 80;
```
```java
var grades = new Object();
grades['egoing'] = 10;
grades['k8805'] = 6;
grades['sorialgi'] = 80;
```
- 위와 같은 객체를 만드는 다른 방법
</br>
</br>

```java
var grades = {'egoing': 10, 'k8805': 6, 'sorialgi': 80};
alert(grades['sorialgi']);
```
- 80 출력
```java
alert(grades.sorialgi);
alert(grades['sorial'+'gi']);
```
- 위와 같은 객체의 속성에 접근하는 다른 방법
</br>

### 객체 (2/3) : 객체와 반복문의 조우
</br>

### < 객체와 반복문 >
- 'for(key in grades) {}'
- grades 객체의 모든 key를 순회하는 반복문
```java
var grades = {'egoing': 10, 'k8805': 6, 'sorialgi': 80};
for(key in grades) {
    document.write("key : "+key+" value : "+grades[key]+"<br />");
}
```
- 출력
```java
key :   egoing value : 10
key :   k8805 value : 6
key :   sorialgi value : 80
```
