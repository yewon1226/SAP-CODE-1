# 반복문
### 반복문 (1/6) : 기본문법
</br>

### while
```java
while (조건){
    반복해서 실행할 코드
}
```
</br>

```java
while(true){
    document.write('coding everybody <br />');
}
```
- 'coding everybody’ 무한대로 출력
- 줄 바꿈 시 `<br/>` 사용

```java
while(false){
    document.write('coding everybody <br />');
}
```
- 출력되지 않음
</br>

---

### 반복문 (2/6) : 반복조건
</br>

### 반복조건
```java
var i = 0;                                        // 종료조건으로 i의 값이 10보다 작다면 true, 같거나 크다면 false가 된다.
while(i < 10){
    document.write('coding everybody <br />');    // 반복이 실행될 때마다 coding everybody <br />이 출력된다. <br /> 줄바꿈을 의미하는 HTML 태그
    i++                                           // i의 값이 1씩 증가한다.
}
```
- coding everybody이 10번 출력
</br>

---

### 반복문 (3/6) : for문
</br>

### for
```java
for(초기화; 반복조건; 반복이 될 때마다 실행되는 코드){
    반복해서 실행될 코드
}
```
</br>

```java
for(var i = 0; i < 10; i++){
    document.write('coding everybody'+i+'<br />');
}
```
- 'coding everybody + i 가 10번 출력
</br>

### 반복문 (5/6) : 반복문의 제어 (break, continue)
</br>

### break
- 반복을 중간에 중단시키고 싶을 때 사용
```java
for(var i = 0; i < 10; i++){
    if(i === 5) {
        break;
    }
    document.write('coding everybody'+i+'<br />');
}
```

```java
coding everybody 0
coding everybody 1
coding everybody 2
coding everybody 3
coding everybody 4
```
</br>

### continue
- 실행은 즉시 중단, 반복은 지속할 때 사용
```java
for(var i = 0; i < 10; i++){
    if(i === 5) {
        continue;
    }
    document.write('coding everybody'+i+'<br />');
}
```

```java
coding everybody 0
coding everybody 1
coding everybody 2
coding everybody 3
coding everybody 4
coding everybody 6
coding everybody 7
coding everybody 8
coding everybody 9
```
</br>

### 반복문 (6/6) : 반복문의 중첩
</br>

### 중첩 반복문 
```java
// 0부터 9까지 변수 i에 순차적으로 값을 할당        
for(var i = 0; i < 10; i++){
    // 0부터 9까지의 변수를 j의 값에 순차적으로 할당
    for(var j = 0; j < 10; j++){    
        // i와 j의 값을 더한 후에 출력
        // String은 숫자인 i와 j의 데이터 타입을 문자로 형태를 변환하는 명령이다. 
        // String()을 제거하고 실행해보면 의미가 좀 더 분명하게 드러날 것이다.
        document.write(String(i)+String(j)+'<br />');
    }
}
```
- `document.write('coding everybody'+i+j+'<br />');` 일 경우 문자가 앞 뒤로 껴있어서 i, j 문자열로 인식
