# 반복문
### 반복문 (1/6) : 기본문법
</br>

### while
```java
while (조건){
    반복해서 실행할 코드
}
```
</br>

```java
while(true){
    document.write('coding everybody <br />');
}
```
- 'coding everybody’ 무한대로 출력
- 줄 바꿈 시 `<br/>` 사용

```java
while(false){
    document.write('coding everybody <br />');
}
```
- 출력되지 않음
</br>

---

### 반복문 (2/6) : 반복조건
</br>

### 반복조건
```java
var i = 0;
// 종료조건으로 i의 값이 10보다 작다면 true, 같거나 크다면 false가 된다.
while(i < 10){
    // 반복이 실행될 때마다 coding everybody <br />이 출력된다. <br /> 줄바꿈을 의미하는 HTML 태그
    document.write('coding everybody <br />');
    // i의 값이 1씩 증가한다.
    i++
}
```
- coding everybody이 10번 출력
</br>

---

### 반복문 (3/6) : for문
</br>

### for
```java
for(초기화; 반복조건; 반복이 될 때마다 실행되는 코드){
    반복해서 실행될 코드
}
```
</br>

```java
for(var i = 0; i < 10; i++){
    document.write('coding everybody'+i+'<br />');
}
```
- 'coding everybody + i 가 10번 출력
</br>

### < case 2 >
```java
document.write('coding everybody 1<br />')
document.write('coding everybody 2<br />')
//중략
document.write('coding everybody 9<br />')
document.write('coding everybody 10<br />')
```
- 비효율적 코드
```java
var i = 0;
while(i < 10){
    document.write('coding everybody '+i+'<br />');
    i++;
}
```
- 효율적 코드
</br>

```java
var i = 0;
while(i < 10){
    document.write('coding everybody '+(i*2)+'<br />');
    i++;
}
```
- coding everybody 뒤에 붙는 숫자를 2의 배수로 변경
