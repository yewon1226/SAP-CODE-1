### 함수 (1/5) : 함수란?
</br>

### 함수
```java
function 함수명( [인자...[,인자]] ){
   코드
   return 반환값
}
```
- function + 함수 이름 + 소괄호
</br>

```java
function numbering(){
    i = 0;
    while(i < 10){
        document.write(i);
        i += 1;
    }   
}
numbering()
```
- 0123456789 출력

### 함수 (3/5) : 입력
</br>

### return
- return 뒤에 따라오는 값을 함수의 결과로 반환. 동시에 함수를 종료시킴
</br>

```java
function get_member1(){
    return 'egoing';
}
 
function get_member2(){
    return 'k8805';
}
 
alert(get_member1());
alert(get_member2());
```
- egoing과 k8805 출력
</br>

```java
function get_member(){
    return 'egoing';
    return 'k8805';
    return 'sorialgi';
}
alert(get_member());
```
- egoing 출력
</br>

### 함수 (4/5) : 출력
</br>

### 인자
- 함수로 유입되는 입력 값
</br>

```java
function get_argument(arg){
    return arg;
}
 
alert(get_argument(1));
alert(get_argument(2));
```
- 1, 2 출력
- 1,2 : 인자, arg: 매개변수(파라미터)
</br>

### 복수의 인자 
```java
function get_arguments(arg1, arg2){
    return arg1 + arg2
}
 
alert(get_arguments(10, 20));
alert(get_arguments(20, 30));
```
- 30, 50 출력
- return 값은 하나만 가질 수 있음
</br>

### 함수 (5/5) : 다양한 정의 방법
</br>

### 함수를 정의하는 또 다른 방법
```java
function numbering (){
    i = 0;
    while(i < 10){
        document.write(i);
        i += 1;
    }   
}
numbering();
```
```java
var numbering = function (){
    i = 0;
    while(i < 10){
        document.write(i);
        i += 1;
    }   
}
numbering();
```
```java
(function (){
    i = 0;
    while(i < 10){
        document.write(i);
        i += 1;
    }   
})();
```
- 익명 함수. 정의와 호출을 같이 함.
