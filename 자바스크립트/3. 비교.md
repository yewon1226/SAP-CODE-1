# 비교

### 비교 (1/4)
- 아래 =는 대입 연산자
```java
a=1
```
- 비교 연산자의 결과는 true나 false 중의 하나(boolean)

```java
alert(1==2)             //false
alert(1==1)             //true
alert("one"=="two")     //false 
```

---

### 비교 (2/4)
</br>

### ==
- 서로 값이 같다면 true 다르다면 false
```java
alert(1==2)             //false
alert(1==1)             //true
alert("one"=="two")     //false 
alert("one"=="one")     //true
```

### ===
- 일치 연산자로 값이 정확하게 같을 때 true 다르면 false
- 서로 같은 수를 표현하고 있더라도 데이터 형이 같은 경우에만 같다고 판단
```java
alert(1=='1');              //true
alert(1==='1');             //false
```
