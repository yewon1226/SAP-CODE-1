# 실전을 위한 SQL 무료 강의
### 실전을 위한 SQL 무료 강의 (1/18) : Select From
</br>

- 특정 테이블에서 원하는 열을 선택해 출력
```sql
SELECT 출력하고 싶은 결과
FROM   테이블 이름
```
</br>

---

### 실전을 위한 SQL 무료 강의 (2/18) : Select * From
</br>

- 해당 테이블의 모든 열을 출력
```sql
SELECT *
FROM   테이블 이름
```
</br>

### 실전을 위한 SQL 무료 강의 (3/18) : Where (1)
</br>

- 조건에 맞는 행만 조회
```sql
SELECT 출력하고 싶은 결과
FROM   테이블 이름
WHERE  조건문
```
</br>

---

### 실전을 위한 SQL 무료 강의 (4/18) : Where (2)
</br>

- 두 개 이상의 조건을 써야할 때는 AND로 연결
```sql
WHERE 나라 = '멕시코' AND 나이 = 30
```
</br>

- 여러 조건 중에 한 조건만 만족해도 될 때는 OR로 연결
```sql
WHERE 나라 = '멕시코' OR 나이 = 30
```
</br>

- 제외하고 싶은 필터가 있을 때는 != 나 <>을 사용
```sql
WHERE 나라 != '멕시코'
WHERE 나라 <> '멕시코'
```
</br>

- 크다는 >, 작다는 <, 크거나 같다는 >=, 작거나 같다는 <=
```sql
WHERE 나이 >= 30 AND 나이 < 40
```
</br>

- 문자는 '', 정수는 그냥!
```sql
WHERE 나라 = '멕시코' OR 나이 = 30
```
</br>

- 같은 열을 이용한 여러 조건은 in / NOT in 과 리스트를 이용
```sql
WHERE 나라 in ('멕시코', '미국', '한국', '영국', '일본')
WHERE 나라 NOT in ('멕시코', '미국', '한국', '영국', '일본')
```
</br>

### 실전을 위한 SQL 무료 강의 (5/18) : Between, Like
</br>

### Where 사용법 (between)
- 범위를 나타낼 때는 BETWEEN을 사용하면 유용
```sql
WHERE 나이 BETWEEN 30 AND 39    // WHERE 나이 >= 30 AND 나이 <= 39 (나이가 30대인 사람만 출력)
```
</br>

- 제외하고 싶은 범위는 NOT BETWEEN을 사용
```sql
WHERE 나이 NOT BETWEEN 30 AND 39    // WHERE 나이 < 30  OR 나이 >= 40 (나이가 30대인 사람만 제외하고 출력)
```
</br>

### Where 사용법 (Like with %)
- 일정 패턴을 찾을 때는 Like/NOT LIKE 연산자가 유용
```sql
WHERE 이름 LIKE 'Tom%'        // 이름이 Tom으로 시작하는 사람을 출력하고 싶다면
WHERE 이름 NOT LIKE '%y'      // 이름이 y로 끝나지 않는 사람을 출력하고 싶다면
WHERE 이름 LIKE '%or%'        // 이름에 or이 어디든 들어있는 사람을 찾으려면 (처음이나 끝이어도 상관없음)
```
</br>

### 실전을 위한 SQL 무료 강의 (6/18) : Order By
</br>

- 조건에 맞는 결과를 특정 열 기준으로 정렬
- ASC: 오름차순, DESC: 내림차순 (생략시 ASC)
```sql
SELECT   출력하고 싶은 결과
FROM     테이블 이름
WHERE    조건문
ORDER BY 열 이름 ASC (DESC)
```
</br>

- 여러 개의 열 이름을 이용해 정렬하고 싶을 때는 콤마로 구분
```sql
ORDER BY 고객가입일 DESC, 고객이름 ASC
```
</br>

### 실전을 위한 SQL 무료 강의 (7/18) : Count
</br>

- 조건에 맞는 데이터 개수(행 수) 출력
- 집계함수 중 하나
- COUNT(*) 혹은 COUNT(열 이름)으로 사용
- 뒤에 AS를 붙이고 이 열의 새로운 이름을 적음
```sql
SELECT  COUNT(*)
FROM    테이블 이름
WHERE   조건문
```
</br>

- COUNT(DISTINCT 열이름)을 사용하면 unique한 행의 수를 셈
- SELECT 결과에서 중복된 값을 제외하고 고유한 값만 반환
```sql
SELECT COUNT(customer_id) as total_customers,
       COUNT(DISTINCT customer_id) as total_unique_customers
FROM Customers
```
</br>

### 실전을 위한 SQL 무료 강의 (8/18) : SUM, AVG, MIN, MAX
</br>

- sql에서 쓰이는 집계함수 중 하나
- SUM : 총 합계를 계산 (빈 행 제외)
- AVG : 평균을 계산 (빈 행 제외)
- MIN : 최소값을 계산
- MAX : 최대값을 계산
```sql
SELECT  SUM(열이름)
FROM    테이블 이름
WHERE   조건문
```
