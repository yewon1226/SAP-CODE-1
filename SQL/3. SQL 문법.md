### <FROM -> WHRE -> GROUP BY -> HAVING -> SELECT -> ORDER BY>
- FROM -> WHERE -> GROUP BY 순으로 실행
- FROM -> GROUP BY 순으로도 작성 가능
- FROM: 테이블 확인
- WHERE: FROM절 테이블을 특정 조건으로 필터링
- GROUP BY: 열 별로 그룹화
- HAVING: 그룹화된 새로운 테이블을 특정 조건으로 필터링
- SELECT: 열 선택
- ORDER BY: 열 정렬
