###  비교 연산자
### 1. = (같음)
- 특정 값과 같은 데이터를 조회할 때 사용
```sql
/* = : 같음 */
SELECT  *
  FROM  CUSTOMER
 WHERE  GENDER = 'MAN';
```
</br>


### 집합연산자 – UNION, UNION ALL
- UNION: 2개 이상 테이블의 중복된 행들을 제거하여 집합
- UNION ALL: 2개 이상 테이블의 중복된 행들을 제거 없이 집합 
* 열 개수와 데이터 타입 일치해야 함
