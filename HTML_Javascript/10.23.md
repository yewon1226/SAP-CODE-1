# 웹 UI 프레임워크 활용한 개발 1-1
- Fiori :  SAP 디자인 가이드라인
- Fiori Launchpad : SAP program/UI5. 다양한 환경에서 업무처리가 가능한 대시보드.
- UI5 : SAP가 권장하는 최적의 UI Application을 만들 수 있는 Javascript 프레임워크 (내부가 Js로 이루어져 있음)

* 참고 사이트 : UI5 SDK 사이트, SAP BTP Trial 사이트
* https://<host>:<port>/sap/public/bc/ui5_ui5/index.html
</br>

### Model, Controller, View 구조 (MVC 패턴)
- UI5는 세 가지 구조로 나뉨.
- UI5는 다양한 브라우저에서 지원.
- View : 화면 담당 (UI)
- Controller : 기능 담당
- Model : 데이터 담당. 데이터의 CRUD 처리. (JSON, oData)
</br>

### SAP에서 주로 사용되는 라이브러리
- sap.ui.comp
- sap.ui.table
- sap.ui.layout
- sap.ui.core
</br>
  
- SAP Business Application Studio (BAS)
- Visual Studio Code
</br>

- SAP UI5 프레임워크의 핵심(JavaScript) 라이브러리를 불러오는 코드
```html
<script
        id="sap-ui-bootstrap"
        src="resources/sap-ui-core.js"
        data-sap-ui-theme="sap_horizon"
        data-sap-ui-resource-roots='{
            "sap.ui5.yui5project01": "./"    // NameSpace.project명
        }'
        data-sap-ui-on-init="module:sap/ui/core/ComponentSupport"
        data-sap-ui-compat-version="edge"
        data-sap-ui-async="true"
        data-sap-ui-frame-options="trusted"
    ></script>
```
