# 11/03 기록

## 🌐 OData 기본 개념 정리

| 용어 | 의미 | 예시 |
|------|------|------|
| **Entity Set** | 여러 개의 데이터를 모아둔 데이터 집합 (즉, 하나의 테이블) | 예: **Orders** 테이블 |
| **Entity** | Entity Set 안에 들어 있는 한 줄의 데이터 (레코드) | 예: 주문번호가 1001인 주문 데이터 |
| **Property** | Entity 안에 포함된 각각의 필드 (열, 속성) | 예: OrderID, CustomerName, OrderDate |
| **Entity Type** | Entity의 전체 구조와 필드 정의 (설계도, 스키마) | 예: Orders 테이블의 필드 구조 정의 |
