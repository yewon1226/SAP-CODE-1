## < ABAP 클래스 정의 기본 구조 >
</br>

- ABAP 클래스는 DEFINITION에서 구조를 선언하고, IMPLEMENTATION에서 동작을 구현
- Public 속성은 외부 접근 가능, Private 속성은 내부 전용이며 READ-ONLY는 외부 수정만 제한
```abap
*************** 클래스 정의 (CLASS DEFINITION) ****************
CLASS class_name DEFINITION.
  PUBLIC SECTION.
  PRIVATE SECTION.
    TYPES: local_type TYPE i.                   " 타입 정의
    CONSTANTS: gc_max TYPE i VALUE 100.         " 상수 정의
  
    DATA: variable1 TYPE local_type,            " 로컬 타입
          variable2 TYPE global_type,           " 글로벌 타입
          variable3 LIKE variable1,             " 기존 변수와 동일 타입
  
          variable4 TYPE i VALUE 10,            " 기본값 지정
          variable5 TYPE string READ-ONLY,      " 읽기 전용 - 외부: 조회 가능, 수정 불가 / 클래스 내부: 둘 다 가능
      
          variable6 TYPE REF TO class_name,      " 클래스 참조
          variable7 TYPE REF TO if_interface,    " 인터페이스 참조
          variable8 TYPE REF TO data.            " 타입 참조
ENDCLASS.

```
```abap
*************** 클래스 정의 (CLASS DEFINITION) ****************
CLASS class_name DEFINITION.
  PUBLIC SECTION.
    " 정적 속성 (Static Attribute)
    " 클래스당 1개만 존재
    CLASS-DATA:
      gv_n_o_vehicles TYPE i.                " 전체 객체가 공유
  PRIVATE SECTION.
    " 인스턴스 속성 (Instance Attribute)
    " 객체 생성 시마다 개별로 생성
    DATA: variable5 TYPE string READ-ONLY.   " 읽기 전용 - 외부: 조회 가능, 수정 불가 / 클래스 내부: 둘 다 가능
ENDCLASS.
```
