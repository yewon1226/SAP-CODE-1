## < ABAP 객체 생성 기본 문법 >
</br>

### 1) ALV 데이터 갱신
- ALV Grid에서 기본 제공되는 표준 버튼(필터, 합계 등)을 화면에서 숨김 → 기능은 유지되고 화면에서만 제거
- 
```abap
DATA: <gt_excluded> TYPE ui_functions.

APPEND cl_gui_alv_grid=><function_code> TO <gt_excluded>.

go_alv_grid->set_table_for_first_display(
  EXPORTING
    i_structure_name     = '<DDIC 구조>'
    it_toolbar_excluding = <gt_excluded>
  CHANGING
    it_outtab            = <itab>
).
```
```abap
DATA: gt_excluded_functions TYPE ui_functions.

APPEND cl_gui_alv_grid=>mc_fc_filter TO gt_excluded_functions.
APPEND cl_gui_alv_grid=>mc_mb_sum    TO gt_excluded_functions.

go_alv_grid->set_table_for_first_display(
  EXPORTING
    i_structure_name     = 'SBOOK'
    it_toolbar_excluding = gt_excluded_functions
  CHANGING
    it_outtab            = gt_bookings
).
```
