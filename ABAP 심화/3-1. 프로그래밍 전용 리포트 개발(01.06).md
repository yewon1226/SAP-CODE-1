## < BDC (Batch Data Creation) >
- 실제 트랜잭션 화면 입력 과정을 BDCDATA에 기록해 프로그램으로 자동 실행하는 방식
</br>

```pgsql
BDCDATA 채움
   ↓
CALL TRANSACTION 'T_CODE'
   USING gt_bdc
   ↓
Dynpro 1 → Dynpro 2 → …
   ↓
SAVE / ENTER / F15 등 OKCODE 실행
   ↓
LEAVE PROGRAM
```
